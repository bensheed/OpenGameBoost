# Assets Directory

Place the following files here:

## icon.ico
The application icon file. Should be:
- Format: ICO
- Size: 256x256 pixels (minimum)
- Multiple sizes recommended: 16x16, 32x32, 48x48, 64x64, 128x128, 256x256

### Creating an Icon
You can create an ICO file using:
- **Online tools:** https://www.icoconverter.com/
- **GIMP:** File → Export As → icon.ico
- **ImageMagick:** `convert logo.png -define icon:auto-resize=256,128,64,48,32,16 icon.ico`

## Recommended Icon Design
- Gaming-themed (controller, rocket, boost symbol)
- Simple, recognizable at small sizes
- Works well on both light and dark backgrounds
- Color scheme: Blue/Cyan (#00d4ff) to match app theme
